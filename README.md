# Bump.tf
Automatically bump TF2Outpost trades

## Usage:

Requires a JSON file called tradeid.json in the same directory as
`tf2op.py` 

Sample:

    {                                                                         
        "uhash":"abcdefg1234567",                           
        "tradeid":[                                                           
            "1234567",                                                       
            "2345678",                                                       
            "3456789",                                                       
            "4567890"                                                        
        ]                                                                     
    }
